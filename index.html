⚙️ 設定 Supabase 連線

首次使用需要設定 Supabase 資料庫連線。設定後資料自動儲存至雲端，跨裝置同步。

1

前往 supabase.com 建立免費帳號並新增專案

2

在 SQL Editor 執行 supabase_setup.sql 建立資料表

3

至 Project Settings → API 複製以下兩個值貼入下方

設定值儲存於 localStorage，不會傳送至任何第三方
